# Compiling with bibliography

    xelatex FA_DM.tex 
    bibtex FA_DM
    xelatec FA_DM.tex
    xelatex FA_DM.tex 

# Simple compilation 
Simply run 

    xelatex FA_DM_.tex

Mac users may want to run 

    xelatex "FA_DM.tex"
    open -a skim  FA_DM.pdf"

Or, alternatively, the run_on_mac.sh script
