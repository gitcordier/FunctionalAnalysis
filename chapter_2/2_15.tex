\textit{Suppose $X$ is an F-space and $Y$ is a subspace of $X$ whose complement is of the first category. Prove that $Y=X$. }%
Hint: \textit{$Y$ must intersect $x+Y$ for every $x\in X$. }%
%
\begin{proof}%
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
% A. Assumptions.
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
Let $\set{E_n}{\counting{n}}$ be such that $X\setminus Y= \bigcup_n E_n$. More specifically assume that $E_n$ is nowhere d%
ense\ie $V_n = X \setminus \closure{E_n}$ is dense in $X$. Now pick an arbitrary $x \in X$. Since the translation by $x$ is a %
self-homeomorphism, $x + V_n$ is open and dense as well. Alternatively, observe that $X= x + X \subset\closure{x+ V_n}$, as a %
special case of \citeFA{1.3 (b)}. %
%
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
% B. Where Baire's theorem is involved.
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
We now apply Baire's theorem twice to establish that %
%
\begin{enumerate}
  \item every intersection $W_n = V_n \cap [x+V_n]$ is dense in $X$,
  \item so is the nonempty intersection $\bigcap_{n=1}^\infty W_n$. 
\end{enumerate}
%
Moreover, the intersection $\bigcap_{n=1}^\infty V_n$ is disjoint from every $E_n$. In summary, %
%
\begin{align}\label{2_15_1}
  w \in \bigcap_{n=1}^\infty W_n \subset \bigcap_{n=1}^\infty V_n \subset Y 
\end{align} 
%
for some $w = w(x)$. %
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
% C. Key ingredient! 
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
Furthermore, $w$ also lies in every $x + V_n$, by (a), which implies %
%
\begin{align}\label{2_15_2}
  w - x \in \bigcap_{n=1}^\infty V_n \subset Y. 
\end{align}
%
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
% D. Put B and C together then conclude. 
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
Finally, (\ref{2_15_1}) and (\ref{2_15_2}) together yield %
%
\begin{align}
  x = w - (w - x) \in Y - Y = Y,
\end{align}
%
as $Y$ is a subgroup of $X$. This establishes that %
%
\begin{align}
  X \subset Y, 
\end{align}
%
since $x$ was arbitrary.
\end{proof}
